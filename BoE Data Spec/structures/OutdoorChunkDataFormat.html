Outdoor Chunk Data Format
(not a game struct)

Size : variable

 

The outdoor chunk data except outdoor strings.

Custom type used
Outdoor record type

Variable array of c-strings containing all the outdoor chunk strings (lengths are set in outdoor.strlens[]).
Ended by the string "From here on, reserved for future use"

String index Usage
0 Area name
1->8 Outdoor areas descriptions
9 Comments
10->99 Outdoor text (specials, etc)
100->107 Outdoor signs
