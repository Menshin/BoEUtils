Monster Record

Size : 0x6C bytes

 

The index of the monster template in the current scenario monsters data.

Value
Index in the scenario.scen_monsters[] : so between 1 and 255, inclusive (first 188 monsters are base game templates).
Value 0 for no monster.

The level of the monster.

Value
Between 0 and 40, inclusive.

C-string containing the monster name.

Value
Any char string, but only the first 19 chars are actually copied on saving.

Current health of the monster.

Value
Any value between 0 and m_health, inclusive.

Max health of the monster.

Value
Any value between 0 and 2500, inclusive.

Current magic points of the monster.

Value
Any value between 0 and m_mp, inclusive.

Max magic points of the monster.

Value
  • No cleric or mage level : 0
  • At least one level of cleric or mage : 14 × level

Armor value of the monster.

Value
Any value between 0 and 50, inclusive.

Skill level of the monster.

Value
Any value between 0 and 40, inclusive.

Attacks of the monster.

Value
Up to 3 attacks in the format : 100 × numbers_of_dice_to_throw + number_of_side_on_a_dice where :
  • numbers_of_dice_to_throw is a integer between 0 and 20 inclusive;
  • number_of_side_on_a_dice is a integer between 1 and 50 inclusive.

The severity of the monster poison, if any.

Value
Any integer between 0 and 8, inclusive.

The morale of the monster poison, if too low it will flee.

Value
Capped at m_morale (can be negative, no lower bound).

The max value of the monster's morale.

Value
  • if (level >= 20) : 10 × level + 10 × (level - 20)
  • else : 10 × level

The id of an item the monster has a chance to drop on death (e.g gland, husk, etc).

Value
Either -1 for no item to drop or any valid item id, so between 0 and 399 inclusive.

Probability of any corpse item to drop.

Value
Percentage, so between 0 and 100.
