Town Data
(not a game struct)

Size : variable

 

The town data except town strings, terrain and talking strings.

Custom type used
Town record type

Structure containing the terrain and creatures for the current town, depending on its size.

Town size in scenario.town_size[] Data structure used
0 big_tr_type
1 ave_tr_type
2 tiny_tr_type

Custom type used
size_tr_type

Variable array of c-strings containing all the outdoor chunk strings (lengths are set in town.strlens[]).

String index Usage
0 Town name
1->16 Town areas descriptions
17->19 Comments
20->119 Town text (specials, etc)
120->139 Town signs

Everything related to talking in this town, except talking strings.

Custom type used
Talking record type

Variable array of c-strings containing all the talking strings for this town (lengths are set in talking.strlens[]).
