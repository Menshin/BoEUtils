Item Storage Shortcut

Size : 0x2C bytes

 

The type of the terrain the item can be put on by the randomizer.

Values
Any integer between -1 (no shortcut) and 255.

The number of the item to be randomly placed.

Value
Any integer between -1 (no item) and 399.

The odd of an item to be placed on a matching tile (percentage).

Value
Percentage, so between 0 and 100.

Specify if the item placed is someone's property or not.

Value
Boolean (inside a short).
