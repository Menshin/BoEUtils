Terrain Type

Size : 0x10 bytes

 

The id of the terrain picture.

Value
Any integer between 0 and 255 (base terrain), 400 and 412 (animated terrain) or greater than 1000 (custom terrain).
Special values (inclusive) Meaning
0 -> 87 Is ground : true
88 -> 120 Is wall : true
121 -> 122 Is ground : true
143 Bed
179 -> 208 Is ground : true
211 -> 212 Is ground : true
215 -> 246 Is ground : true
400 -> 412 Animated
404 Lava
1000 -> 1999 Custom picture
>= 2000 Custom animated picture
