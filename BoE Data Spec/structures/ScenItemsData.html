Scenario Item Data

Size : 0x9920 bytes

 

Items templates for the current scenario.

Custom type used

Item Record

Contains informations of the size and visibility of the scenario towns.

Description

Initialization values for various scenario variables (some needed to properly parse the scenario file).

  • town_size[200] : size of each town in the scenario (even for unused towns).
  • town_hidden[200] : visibility of each town in the scenario (even for unused towns).
